# CSS의 우선순위

- HTML 문서의 해석방법은 인터프리터 방식이기 때문에 태그와 가장 가까운 효과가 우선순위가 높다.
- 선택자 사이의 우선순위
    - `!important > Inline style > id > class > tag > *`
		
## CSS를 적용시키는 방법

- **Embedding Style**
    - <style>태그를 사용하여 문서에 직접 정의하는 방식
- **Inline Style**
    - 적용하고자 하는 태그에 style속성을 이용하여 직접 정의하는 방식
- **External Style**
    - 별도의 파일을 만들어서 해당문서에 링크시켜 사용하는 방식
