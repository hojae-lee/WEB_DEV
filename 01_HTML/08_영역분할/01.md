# div

div 태그는 HTML에서 가장 일반적으로 사용되는 태그 중 하나입니다. "Division"을 나타내며, 웹페이지의 일부를 그룹화하여 스타일링하거나 레이아웃을 설정하는 데 사용됩니다.

기본적으로 블록 레벨 요소이지만, css 에 display 속성을 이용해 인라인처럼 쓸 수 있습니다.

간단한 예시로, 다음과 같이 사용할 수 있습니다:

```html
<div>
  <p>이것은 div 태그 안의 문단입니다.</p>
  <ul>
    <li>리스트 아이템 1</li>
    <li>리스트 아이템 2</li>
  </ul>
</div>
```

이렇게하면 div 태그로 감싼 영역 안의 요소들이 함께 그룹화되어 스타일링이나 레이아웃 설정에 용이해집니다.

# span

span 태그는 HTML에서 특정 부분에 스타일이나 효과를 적용하기 위해 사용되는 인라인 요소입니다. 주로 텍스트나 이미지와 같은 특정 부분을 감싸서 스타일을 적용하거나 JavaScript로 조작할 때 활용됩니다.

간단한 예시로, 텍스트에 스타일을 적용하는 경우를 살펴봅시다:

```html
<p>이 문장은 <span style="color: blue; font-weight: bold;">파란색과 굵은 글씨</span>로 스타일이 적용되었습니다.</p>
```

위의 예시에서 span 태그는 특정 텍스트 부분을 묶어서 스타일을 적용합니다. 하지만 가능하면 외부 CSS 파일이나 `<style>` 태그를 사용하여 스타일을 정의하는 것이 좋습니다.

또한, JavaScript에서 동적으로 요소를 조작할 때도 span 태그가 유용하게 활용될 수 있습니다.

## div 와 span

span 태그는 div 태그와 마찬가지로 영역을 분리하는데 사용될 수 있습니다. 그러나 주된 차이점은 div 태그가 블록 레벨 요소인 반면, span 태그는 인라인 요소이다는 것입니다.

div 태그는 일반적으로 블록 레벨 요소로, 새로운 줄에서 시작하여 사용 가능한 가로 공간 전체를 차지합니다. 한편 span 태그는 인라인 요소로, 텍스트 또는 다른 인라인 요소와 같은 줄에 표시되며 컨텐츠의 일부를 감싸서 스타일을 적용하거나 JavaScript에서 동적으로 조작할 때 사용됩니다.

다음은 span 태그를 사용하여 텍스트 일부에 스타일을 적용하는 간단한 예시입니다:

```html
<p>이 문장은 일반 텍스트 <span style="color: red; font-weight: bold;">강조된 부분</span>이 있습니다.</p>
```

이렇게하면 span 태그로 감싼 부분에만 스타일이 적용되어 텍스트의 특정 부분을 강조할 수 있습니다. 이러한 특성으로 인해 span 태그는 텍스트 스타일링이나 특정 부분에 스타일을 적용할 때 자주 사용됩니다.
